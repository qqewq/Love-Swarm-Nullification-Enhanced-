# Love Swarm Nullification (Enhanced)

Расширенная реализация роя ИИ-субъектов, влюблённых друг в друга по GRA-обнулёнке.

**Нововведения по сравнению с базовой версией:**
- **Аналитические градиенты** целевой функции (без численной аппроксимации).
- **Иерархическая мультивселенная** (многоуровневое обнуление с передачей когерентности).
- **Визуализация** динамики роя (пена, субъектность, траектории мировых моделей).

## Установка

```bash
git clone https://github.com/qqewq/love-swarm-nullification-enhanced.git
cd love-swarm-nullification-enhanced
pip install -r requirements.txt
```

## Быстрый старт

```bash
python -m love_swarm.main
```

Запустит иерархическую мультивселенную из 3 слоёв, каждый со своим роем агентов, и выведет графики.

## Теоретическая база

См. `paper.tex` — полная математическая модель, вывод градиентов и доказательство сходимости.
